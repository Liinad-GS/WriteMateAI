# Chrome Web Store Screenshot Shot List

Last updated: March 28, 2026

## Goal

Prepare 3 to 4 clean screenshots that make the extension understandable in a few seconds.

## Screenshot 1: Translate In Context

Show:

- selected message text on a chat or email page
- floating WriteMate AI trigger
- open widget on the `Translate` tab
- translated result visible

Suggested caption idea:

`Translate selected text without leaving the page`

## Screenshot 2: Grammar Fix

Show:

- short email or message draft with mistakes
- `Grammar` action selected
- corrected result visible
- `Replace text` button visible if possible

Suggested caption idea:

`Fix grammar and spelling in seconds`

## Screenshot 3: Tone Rewrite

Show:

- original text that sounds too blunt or awkward
- `Tone` tab selected
- tone option visible
- rewritten result visible

Suggested caption idea:

`Rewrite the same message in a better tone`

## Screenshot 4: Improve Or Feedback

Choose one:

- `Improve` flow with a clearer rewritten result
- or the daily limit modal with feedback form visible

Suggested caption idea:

`Improve clarity for messages, chats, and emails`

Alternative caption idea:

`Collect feedback and guide users when free usage ends`

## Practical Tips

- Use a clean browser window with no distracting tabs or bookmarks
- Keep the selected text short enough to read in the screenshot
- Make sure the widget, result area, and action buttons are readable
- Avoid showing private messages, real emails, or sensitive personal data
- Keep the same browser zoom level across screenshots
- Prefer desktop screenshots unless mobile behavior is a selling point

## Recommended Order In The Store

1. Translate
2. Grammar
3. Tone
4. Improve or feedback modal
