WriteMate AI

WriteMate AI is a Chrome extension for AI-assisted translation and writing directly inside web pages. It combines a Manifest V3 extension, a Node.js backend proxy, streaming LLM responses, usage limits, analytics, and feedback collection.

  • Translate selected text without switching tabs
  • Fix grammar, improve clarity, and adjust tone
  • Stream AI output as one progressively growing result
  • Use a backend proxy with OpenAI, BigQuery analytics, and feedback webhooks
